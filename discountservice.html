DiscountService API

DiscountService is used to manage products. You can add, update and find products in addition to managing follow up sequences, tags and action sets.

DiscountService.addFreeTrial

returns the inventory

Required Parameters

Parameter Name Type Definition
name string name of the free trial
description string description for free trial
freeTrialDays int number of days free trial last
hidePrice int option to show or hide price
subscriptionPlanId int Id of the subscription

  <params>
    <param>

    </param>
    <param>
      <value><string>name</string></value>
    </param>
    <param>
      <value><string>description</string></value>
    </param>
    <param>
