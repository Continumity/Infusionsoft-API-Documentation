WebFormService API

The WebFormService is used to work with web forms within Infusionsoft.

WebFormService.getMap

This retrieves the web form names and Id numbers from the application.

Required Parameters

Parameter Name Type Definition
privateKey string Your Infusionsoft API key

Sample Request

<?xml version='1.0' encoding='UTF-8'?>
<methodCall>
  <methodName>WebFormService.getFile</methodName>
  <params>
    <param>
      <value><string>privateKey</string></value>
    </param>
    <param>
      <value><int>fileUID</int></value>
    </param>
  </params>
</methodCall>

Sample Response

<?xml version='1.0' encoding='UTF-8'?>
<methodResponse>
  <params>
    <param>
      <value>fileData</value>
    </param>
  </params>
</methodResponse>

PHP iSDK Sample

$fileId = 64;
$myUrl = $app->getFile($fileId);
WebFormService.getDownloadUrl

This method will return a string of the download URL for the given file.

Required Parameters

Parameter Name Type Definition
FileId int The ID of the file url to be returned.

Sample Request

<?xml version='1.0' encoding='UTF-8'?>
  <methodCall>
    <methodName>WebFormService.getDownloadUrl</methodName>
    <params>
      <param>
        <value><string>privateKey</string></value>
      </param>
      <param>
        <value><int>fileUID</int></value>
      </param>
    </params>
  </methodCall>

Sample Response

<?xml version="1.0" encoding="UTF-8"?>
<methodResponse>
  <fault>
    <value>
      <struct>
        <member>
          <name>faultCode</name>
          <value><i4>3</i4></value>
        </member>
        <member>
          <name>faultString</name>
          <value>[Unexpected]Error getting download url: Access denied</value>
        </member>
      </struct>
    </value>
  </fault>
</methodResponse>

PHP iSDK Sample

$fileId = 64;
$myUrl = $app->getDownloadUrl($fileId);
WebFormService.uploadFile

This method uploads the file to Infusionsoft. The optional contactId parameter is used to place the file in a specific contactâ€™s filebox.

Required Parameters

Parameter Name Type Definition
FileName string The name of the file to be uploaded
Base64EncodedData string A string that is 64 base encoded.
ContactId (optional) int ID of the contact record to add the file to.

Sample Request

<?xml version='1.0' encoding='UTF-8'?>
<methodCall>
  <methodName>WebFormService.uploadFile</methodName>
    <params>
      <param>
        <value><string>privateKey</string></value>
      </param>
      <param>
        <value><int>contactID</int></value>
      </param>
      <param>
        <value><string>fileName</string></value>
      </param>
      <param>
        <value><string>RGVycE15RGVycA==.......base64EncodedFile</string></value>
      </param>
    </params>
  </methodCall>

Sample Response

<?xml version="1.0" encoding="UTF-8"?>
<methodResponse>
  <params>
    <param>
      <value><i4>482</i4></value>
    </param>

WebFormService.replaceFile

This method will return a string of the download URL for the given file.

Required Parameters

Parameter Name Type Definition
FileId int ID of the file to be replaced.
Base64EncodedData string New string of data.

Sample Request

<?xml version='1.0' encoding='UTF-8'?>
<methodCall>
  <methodName>WebFormService.replaceFile</methodName>
  <params>
    <param>
      <value><string>privateKey</string></value>
    </param>
    <param>
      <value><int>fileID</int></value>
    </param>
    <param>
      <value><string>base64EncodedFile</string></value>
    </param>
  </params>
</methodCall>

Sample Response

<?xml version='1.0' encoding='UTF-8'?>
<methodResponse>
  <params>
    <param>
      <value><boolean>1</boolean></value>
    </param>
  </params>
</methodResponse>
